## English

### What is stored in the database

The purpose of this database is to store various information about fictional star systems, from the composition of a celestial body's ring to the types of stars present in each system.
The data source is a space simulator called Elite Dangerous, which is why some entities have a high level of detail in the stored data, such as the percentage of each element in a planet's atmosphere.

## Português (Brasil)

### O que está armazenado no banco de dados

O objetivo deste banco de dados é armazenar diversas informações sobre sistemas estelares fictícios, desde a composição do anel de um corpo celeste até quais tipos de estrelas estão presentes em cada sistema.
A fonte dos dados é um simulador espacial chamado Elite Dangerous, por causa disso algumas entidades possuem grande detalhamento dos dados armazenados, como a porcentagem de cada elemento na atmosfera de um planeta.
